I made the first version of this program way back in 2018 when I was curious about my map winrates. 
I have iterated on it several times over the years and am now making it public.

Program Features:
-Allows for the creation of 'stat profiles' (season 1 2026, season 2 2026, etc...) to neatly save data to its own file for future review.
-Allows for the creation of 'profiles' to separate multiple accounts (main account, alt account, 5v5, 6v6).
-Document your current session and save the data to a 'profile' and 'stat profile'
-View stats for saved sessions.
-Add new maps and game modes with ease when new content comes out.
-Exporting stats with the click of a button to a .CSV file ready to import to google sheets or similar programs.
